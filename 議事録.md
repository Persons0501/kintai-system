# 勤怠管理システム - 開発議事録

## 日付: 2026-03-05

---

## 1. 要件（ユーザー指示）

- 派遣会社で従業員用の簡易的な勤怠システムを作りたい
- 出社ボタンと退勤ボタンのみ
- 派遣従業員が外国人人材のため、めっちゃ簡易的なものでいい
- 打刻データ：日付、出社、退勤、労働時間、休憩時間
- CSV もしくは Excel データでダウンロードできるようにしたい
- 勤務先がわかるようにしたい
- ログインは ID とパスワード（従業員にランダムに配布）

---

## 2. 提案・決定事項

### 技術スタック

| 構成要素 | 技術 |
|----------|------|
| バックエンド | Python (Flask) |
| データベース | SQLite（軽量・セットアップ不要） |
| フロントエンド | HTML/CSS/JavaScript（シンプルなUI） |
| データ出力 | openpyxl（Excel出力）/ CSV（csv module + BOM付きUTF-8） |

### 画面構成（3画面）

1. **ログイン画面** — ID・パスワード入力のみ（日英バイリンガル）
2. **打刻画面（従業員用）** — 大きな「出勤」「退勤」ボタン + リアルタイム時計 + 本日の打刻記録
3. **管理画面（管理者用）** — 本日の出勤状況・従業員管理・勤怠データ閲覧/修正・CSV/Excelダウンロード

### 休憩時間

- **自動計算**（労働基準法準拠）
  - 6時間超〜8時間以下 → 休憩45分
  - 8時間超 → 休憩60分

## 4. 起動手順（Windows PowerShell）

```powershell
cd "$env:USERPROFILE\OneDrive\Desktop\勤怠システム"
py -m venv venv
.\venv\Scripts\Activate.ps1
pip install -r requirements.txt
python app.py
```

- ブラウザで `http://localhost:5000` にアクセス

> `Activate.ps1` でエラーが出る場合：
> ```powershell
> Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser

## 6. 機能一覧

### 従業員向け
- ID・パスワードでログイン
- 大きな「出勤」「退勤」ボタンで打刻（いつでも押せる、上書き可能）
- 上書き時は確認ダイアログ表示（誤押し防止）
- 本日の出勤・退勤時刻をリアルタイム表示
- リアルタイム時計表示
- 勤務先の表示
- 日英バイリンガル表示（外国人対応）

### 管理者向け
- **本日の出勤状況一覧**（全従業員の出勤/退勤/未出勤がパッと見える）
- 従業員の登録（名前・勤務先 → ID/PW自動生成）
- 従業員情報の編集（名前・勤務先の変更）
- パスワード再発行（リセットボタン）
- 従業員の削除（確認ダイアログ付き）
- 勤怠データの閲覧（年月・従業員・勤務先でフィルタ）
- 勤怠データの手動修正・追加（打刻忘れ対応）
- CSV / Excel ダウンロード（フィルタ条件反映）
- 出力項目：日付、名前、勤務先、出勤、退勤、休憩時間、労働時間

## 8. 備考・注意点

- WSL環境ではpip/venvが未インストールだったため、Windows側のPython（3.14.3）を使用
- CSVはBOM付きUTF-8で出力（Excelで文字化けしないように対応済み）
- 管理者パスワード（admin123）は本番運用前に変更推奨
- `host='0.0.0.0'`で起動するため、同一ネットワーク内の端末からアクセス可能
